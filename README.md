## Projects

### AquaTrace

AquaTrace is a vessel-tracking system designed to monitor and track maritime activity around the Davao region and its ports. It provides real-time or systematic tracking of ships, helping improve visibility of vessel movements for better monitoring, coordination, and maritime awareness.

The system is built to support safer and more efficient port operations by organizing vessel data in a clear and accessible way.

**Technologies:** Tracking Systems, Monitoring, Database

---

### EcoTour+

A web-based eco-tourism platform designed to showcase destinations, promote sustainable travel, and help users explore natural attractions in an organized and user-friendly way.

It includes destination listings, categories, and information pages that highlight environmental beauty while encouraging responsible tourism and conservation awareness.

**Technologies:** Tourism, Sustainability, Web Application

### Disaster Relief Goods Distribution System

An algorithmic solution using the Knapsack 0/1 Algorithm to optimize goods distribution during disaster response, maximizing aid value per beneficiary under strict resource constraints.
